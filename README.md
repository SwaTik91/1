# MishpuchaTech

**MishpuchaTech** (от ивритского "мишпаха" - семья) - концепт закрытой цифровой платформы для горско-еврейской общины.

## Слоган

**Ваша семья. Ваше сообщество. Ваше цифровое пространство.**

## Идея проекта

Сегодня значительная часть общинной жизни держится на WhatsApp-чатах, личных связях и неформальной координации. Это неудобно, небезопасно и плохо масштабируется. MishpuchaTech предлагает единый цифровой дом, где семья, история, события, взаимопомощь и локальный нетворкинг объединены в одном приватном пространстве.

## Что входит в первую версию репозитория

- Одностраничный адаптивный landing page на русском языке
- Четко сформулированное позиционирование проекта
- Блоки с ключевыми функциями продукта
- Аргументация ценности и сетевого эффекта
- Описание базовой бизнес-модели

## Ключевые функции платформы

1. **Профиль и личные связи**
   - Профиль с указанием семьи, рода и места происхождения
   - Инвайт-механика для доступа только "своих"

2. **Интеллектуальное семейное древо**
   - Конструктор семейного древа
   - Автоматическое предложение объединения ветвей при совпадениях

3. **Приватные группы**
   - Группы для семьи, событий, соседей, выпускников
   - Общий календарь дней рождения и общинных праздников

4. **Лента сообщества**
   - Новости о рождении, свадьбах, бизнесах и событиях

5. **Найди земляка**
   - Геолокационная функция по взаимному согласию

6. **Система взаимопомощи "Цдака"**
   - Безопасные и прозрачные сборы внутри сообщества

## Бизнес-модель

- **Freemium**: базовые возможности бесплатны
- **Премиум**: расширенное хранилище и аналитика древа
- **Комиссия**: 2-3% со сборов внутри платформы
- **Партнерства**: продвижение проверенных бизнесов общины

## Структура проекта

- `index.html` - основная landing page
- `styles.css` - стили и адаптивная сетка
- `script.js` - мобильная навигация и плавность взаимодействия
